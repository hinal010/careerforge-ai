# careerforge-ai
About AI-powered resume generator using NLP to create job-specific, professional resumes from user skills and experience. Fast, accurate, and ATS-friendly.
Installation & Setup
1️⃣ Clone Repository
git clone https://github.com/hinal010/careerforge-ai.git
cd careerforge-ai
2️⃣ Create Virtual Environment
python -m venv venv
source venv/bin/activate   # Linux / Mac
venv\Scripts\activate      # Windows
3️⃣ Install Dependencies
pip install -r requirements.txt
4️⃣ Run Backend Server
uvicorn main:app --reload
5️⃣ Open Application
Visit: http://127.0.0.1:8000
